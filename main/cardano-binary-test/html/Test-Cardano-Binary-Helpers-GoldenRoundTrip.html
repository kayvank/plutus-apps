goldenTestCBOR :: forall a. (FromCBOR a, ToCBOR a, Eq a, Show a, HasCallStack) => a -> FilePath -> Property Source #

Check that the encode and decode function of the Bi instances work as expected w.r.t. the give reference data, this is, given a value x::a, and a file path fp:

  • The encoded data should coincide with the contents of the fp.
  • Decoding fp should give as a result x

Round trip test a value (any instance of FromCBOR, ToCBOR, and Show classes) by serializing it to a ByteString and back again and that also has a Show instance. If the a type has both Show and Buildable instances, it's best to use this version.
