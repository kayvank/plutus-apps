Cardano.Wallet.DB.Store.Meta.Model

Description

Pure, low level model for a collection of "meta transactions", i.e. additional data (TxMeta) that the wallet stores for each transaction. Meta transactions are encoded "as" expressed in DB tables.

data ManipulateTxMetaHistory Source #

Verbs for TxMeta changes that can be issued independently from the transaction store.

Constructors

PruneTxMetaHistory TxId

Remove a meta if it is not in the ledger.

AgeTxMetaHistory SlotNo

Change the state of any meta to Expired if the given slot is equal or after its expiration slot.

RollBackTxMetaHistory SlotNo

Remove all incoming transactions created after the given slot and mark all outgoing transactions after the given slot as Pending.
